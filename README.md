# Skill Swap Tracker

Skill Swap Tracker is a MERN Stack web application that helps users exchange skills with each other. Users can create profiles, add the skills they can teach and the skills they want to learn, find compatible matches, send swap requests, and join online meetings after a request is accepted.

## Features

- User Registration & Login (JWT Authentication)
- User Profile Management
- Add and Update Skills
- Smart Skill Matching
- Send and Receive Skill Swap Requests
- Accept or Reject Requests
- Automatic Meeting Link Generation
- Online Video Meeting Integration
- Responsive User Interface

## Tech Stack

### Frontend
- React.js
- Vite
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Installation

### Clone Repository

```bash
git clone https://github.com/Anas2634/skill-swap-tracker
```

### Install Dependencies

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm run dev
```
